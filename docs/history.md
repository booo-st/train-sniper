# 개발 히스토리

## 2026-06-22 — UI 개편 및 기능 안정화

- **잡 영속성**: 서버 재시작 후에도 작업 목록 유지, "재개" 버튼으로 재시작
- **예약 완료 잡 재개**: 완료된 잡에도 재개 버튼 추가
- **과거 날짜 차단**: 생성 시 및 워커 시작 시 과거 날짜 감지 → 즉시 거부/중지
- **UI 개선**: 삭제 확인 모달, 로딩 피드백, 작업 그룹 분리(실행중/중지/완료), 진행 바, 패널 접기
- **좌석 체크박스**: 드롭다운 → 일반실/특실 각각 체크박스, 둘 다 선택 시 일반실 우선
- **역 드롭다운**: 출발/도착 텍스트 입력 → KTX/SRT별 역 목록 드롭다운
- **최소 조회 간격**: 10초 → 1초
- **텔레그램 테스트 에러 전파**: 실패 시 이제 토스트에 실제 오류 메시지 표시
- **텔레그램 네트워크 이슈 진단**: `api.telegram.org` 네트워크 레벨 차단 → VPN 필요
- **GitHub 레포 생성**: `booo-st/train-sniper`에 초기 커밋 및 푸시
