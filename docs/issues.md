# 이슈

## 알려진 한계

- **ngrok 무료 플랜 URL 변경**: 프로세스 재시작마다 URL이 바뀜 → 가족에게 재공유 필요
- **세션 미영속**: 서버(`app.py`) 재시작 시 로그인 세션 초기화 → 재로그인 필요
- **Mac 의존**: `caffeinate`가 macOS 전용이라 다른 OS에서는 잠자기 방지 기능 동작 안 함
- **텔레그램 네트워크 차단**: 일부 네트워크(회사망 등)에서 `api.telegram.org` 접속 자체가 막혀 알림 불가 → VPN 필요. 테스트 버튼이 성공처럼 보여도 실제 미전송이었던 원인 (2026-06-22 진단)
